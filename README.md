# Portfolio

A developer portfolio built with **Next.js** (App Router), **TypeScript**, **TailwindCSS**, and **shadcn/ui**. Supports **pt-BR** and **en-US** languages.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Stack

- Next.js (App Router)
- React 19
- TypeScript
- TailwindCSS
- shadcn/ui
- Lucide Icons
- i18n (pt-BR / en-US)
